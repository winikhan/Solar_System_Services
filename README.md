# SolarCare - Solar System Services Website

SolarCare is a professional website designed to showcase solar panel services including cleaning, inspection, repair, and small installations. The website provides users with clear information about available services, pricing, and contact details, all in a simple and user-friendly interface.



 Project Overview

The SolarCare website includes:

- Home Page: A hero banner with a welcoming message, tagline, and a call-to-action button to explore services.  
- Featured Services Section: Displays main services with images, descriptions, and price ranges for different setups.  
- Navigation Bar: Provides links to Home, About, Services, and Contact pages for easy browsing.  
- Footer: Includes email, phone number, and office address for quick contact.

---

Technologies Used

- HTML5 for structure  
- CSS3 for styling and layout  
- Responsive design for all device sizes
